## Neural Networks 
Neural Networks lab exercises, NTUA, 2019-2020

## Content
:rocket: **Lab1**   
Supervised Learning working on UCI Machine Learning Repositories; Quality Assessment of Digital Colposcopies Data Set and Nomao dataset.
